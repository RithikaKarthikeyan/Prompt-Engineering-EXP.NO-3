### EX.NO:3  Exploring the different types of prompts- Straightforward Prompts - Tabular Format Prompting - Missing Word Prompting - Preceding Question Prompting.

Aim: To write the prompts for the following prompt types for 2 different scenarios of Each type.

1.Straightforward Prompts

2.Tabular Format Prompting

3.Preceding Question Prompting

4.Missing Word Prompting

Explanation - Any one use case from Unit 5 or any 2 own use cases with the unit 2 Prompt type

Procedure:

Write 2 different prompts for different use cases, refer the following Examples

Example:

Straightforward Prompts:

• "Define photosynthesis in one sentence."

"List three advantages of electric vehicles."

Tabular Format Prompting:

• "Compare and contrast AC and DC current in a table."

• "Provide a table listing five programming languages, their paradigms, and one use case each."

Preceding Question Prompting:

• "Why is climate change a global concern? Explain how greenhouse gases contribute to global warming."

• "How do vaccines work? Describe the process of immunization in simple terms."

Missing Word Prompting:

• "The capital of France is ____."

• "In photosynthesis, plants absorb sunlight to produce ____."

## OUTPUT:

#### 1. Straightforward Prompts

Prompt 1:
Explain how GPS technology is used for navigation.

Answer:
GPS uses satellites to determine the exact location of a device. It helps users find directions, track locations, and navigate roads using smartphones or navigation systems.

Prompt 2:
List three applications of robotics in industry.

Answer:

Assembly line automation

Welding in automobile industries

Packaging and material handling

#### 2. Tabular Format Prompting

Prompt 1:
Create a table comparing HTTP and HTTPS.

| Feature    | HTTP            | HTTPS                   |
| ---------- | --------------- | ----------------------- |
| Security   | Less secure     | More secure             |
| Encryption | No encryption   | Uses SSL/TLS encryption |
| Usage      | Normal websites | Secure websites         |


Prompt 2:
Provide a table of three types of machine learning.

| Type          | Description            | Example           |
| ------------- | ---------------------- | ----------------- |
| Supervised    | Uses labeled data      | Spam detection    |
| Unsupervised  | Finds hidden patterns  | Customer grouping |
| Reinforcement | Learns through rewards | Game AI           |

 
#### 3. Preceding Question Prompting

Prompt 1:

Why is data privacy important? Explain how personal data can be protected.

Answer:

Data privacy protects personal information from misuse. It can be protected by using strong passwords, enabling two-factor authentication, and avoiding unsafe websites.

Prompt 2:

How does artificial intelligence help in healthcare?

Answer:

AI helps doctors analyze medical images, detect diseases early, and assist in accurate diagnosis and treatment planning.

#### 4. Missing Word Prompting

Prompt 1:

The technology that connects everyday devices to the internet is called ______.

Answer:

Internet of Things (IoT)

Prompt 2:

The device that directs data packets between networks is called a ______.

Answer:

Router
